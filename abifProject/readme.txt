Сборка библиотеки
А. Что нужно скачать\установить
 - SWIG - отсюда http://www.swig.org/download.html взять архив swigwin-2.0.11, распаковать куда-нибудь
 - отсюда http://rubyinstaller.org/downloads/ - взять ruby 1.9.3 и Devkit, установить; пути внутри девкита к исполняемым файлам 
 прописать в переменную окружения PATH (либо при установке, либо ручками)
Б. Последовательность действий 
 1. Берем файл AbifReader.i, передаем его SWIG:
 ..\..\swigwin-2.0.11\swig.exe -c++ -ruby AbifReader.i
 путь к swig указываем тот, куда распаковали, не забыть указать ключи -c++ -ruby 
 В результате должен появиться файл AbifReader_wrap.cxx
 2. Запускаем makegen.rb
 ruby makegen.rb
 - он должен сгенерировать Makefile.
 3. Запускаем команду make
 Если все ок, она сгенерирует библиотеку с расширением .so. Если нет, выдаст сообщения об ошибках.
